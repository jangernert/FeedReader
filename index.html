FeedReader RSS desktop client



FeedReader is a modern desktop application designed to complement existing web-based RSS accounts. It combines all the advantages of web based services like synchronisation across all your devices with everything you expect from a modern desktop application.

Installation


  • Ubuntu / elementary OS
  • Arch Linux
  • Solus OS
  • Fedora

    • To install FeedReader in Ubuntu 15.04 (or newer) or in elementary OS Freya, paste these three commands into your terminal:

    sudo add-apt-repository ppa:eviltwin1/feedreader-stable
    sudo apt-get update
    sudo apt-get install feedreader

    Note that FeedReader requires that you have Gtk 3.12 or newer installed. FeedReader will not work with Ubuntu-based systems that do not satisfy this criteria. To check which version of Gtk is running on your system, open up your terminal and type: 

    dpkg -l libgtk-3-0

    FeedReader is available as an AUR package.

    FeedReader is already packaged in the software-center of Solus.

    Available in the default repository of Fedora 21 and newer.
